Responses

Creating and removing responses will soon be possible. In the meantime, please use the existing commands.

Each row represents a response, which is composed of a command and a body.

The cells in the Commands columns are the names of the commands that trigger the response of the corresponding row. So a response containing both "hello" and "world" in the "command" column would be executed by !hello or !world.

Similarly, the cells in the Responses column is the text printed to the chat when the response of the corresponding row is triggered. So a response containing both Hello, world! and Howdy! would be randomly selected when the corresponding commands are triggered.
